# Teslo API

## Description

API for Teslo application. This API is built with NestJS and TypeORM.

## Environment Variables

```bash
# Database
cp .env.template .env
```

## Up database with Docker

```bash
docker compose up -d
```

## Installation

```bash
npm install
```

## Execut seed

```
http://localhost:3000/api/seed
```

## Running the app

```bash
# development
npm run start:dev
```
